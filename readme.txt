Special Box for Content

== Description ==

This plugin helps you to add a box left or right of your content that can stand out. You can add inside the shortcode images or text.

Easy to use simply to edit!



== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto settings -> Special Box Styling and set the default values or your own 
4. Usage: [boxleft] Sample Content [/boxleft] or [boxright] Sample Content [/boxright] on your post or page editor.
